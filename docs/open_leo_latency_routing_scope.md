# Open LEO Latency Routing Scope

## Working Title

Temporal Forecasting, Graph-Based Representation, and Predictive Optimization on Open LEO Network Measurements

## Core Question

Can short-horizon prediction and temporal graph structure improve decision quality over reactive baselines on open LEO measurement data?

## Mandatory Comparisons

- Predictive optimizer vs reactive optimizer
- Predictive + graph vs predictive-only
- Predictive + graph vs graph feature ablation
- Robustness under forecast noise

## Reviewer Risks To Address

- limited time window sensitivity,
- graph feature redundancy,
- prediction error robustness,
- computational overhead.
